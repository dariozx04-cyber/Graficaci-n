 light_position = [3.0, 2.0, 3.0, 1.0]

light_diffuse = [1.0, 1.0, 1.0, 1.0]

 light_ambient = [0.3, 0.3, 0.3, 1.0]

## light_position

Define la posición de la luz.

x = 3.0

y = 2.0

z = 3.0

w = 1.0

## light_diffuse

Define el color e intensidad de la luz principal.

Es RGBA:

rojo = 1.0

verde = 1.0

azul = 1.0

alpha = 1.0

## light_ambient

Define la luz ambiental.

La luz ambiental ilumina todo un poco aunque no reciba luz directa.

0.3 = iluminación suave.

## light_direction

Eso define una luz direccional.

x = 1.0
y = -1.0
z = 1.0

significa: (1,-1,1)
